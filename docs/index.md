## 这是什么？

把同一个真实开发需求（给 Agent 加激活/停用功能，5 条业务规则）丢给三个不同的 AI 编程辅助工具，**每个工具跑一次独立的 Claude Code session**，看它们在"想清楚再动手"这件事上各有什么打法、最终产出什么样的代码、消耗多少 token、留下什么样的文档。

**这不是评测。** 是一份 N=1 的体验笔记，带数据、带截图、带踩坑记录，也带主观感受。建议先看[速览](/overview/)，再决定要不要深入。
