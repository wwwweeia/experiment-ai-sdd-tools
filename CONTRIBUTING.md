# 贡献指南

这份仓库是一份开放的"工具体验馆"，欢迎补充新一轮、修正错误、提出新对比维度。

---

## 主要贡献机会：跑一轮新工具

仓库保留了 3 个占位目录（`04-tool-tbd/`、`05-tool-tbd/`、`06-tool-tbd/`），等待新工具补充。

理想的候选工具：
- AI 编程辅助 / 规范驱动开发类工具（Cursor 的 Rules、Cline、Aider、Continue、Roo Cline 等）
- 跟 Claude Code 生态相关的方法论工具（其他类似 Superpowers 的插件）
- 任何你觉得"应该跟这三个工具放在一起对比"的东西

### 怎么开始

1. **Fork 本仓库**
2. **跑完一轮实验** —— 完整流程见 [`docs/HOW-TO-REPLICATE.md`](docs/HOW-TO-REPLICATE.md)
3. **保留以下产出**：
   - 重命名后的轮次目录（如 `04-cursor/`），含代码 + 工具产出 + 截图
   - 该轮的 AI 自评笔记（保存在轮次目录内，命名 `observation-notes.md`）
   - 最终人评笔记（保存在 `notes/round4-cursor.md`，结构参考前三轮）
   - 更新 `notes/comparison.md` 的概览表
   - 更新 `README.md` 的 30 秒速览表
4. **提 Pull Request**，简短描述：
   - 工具名 + 版本号
   - 你的整体感受（一句话）
   - 跟现有三轮的最大差异点

## 笔记格式规范

为保持横向可比性，请参考前三轮的笔记结构。**关键章节必须有**：

| 章节 | 内容 |
|------|------|
| 工具速览盒子（顶部） | 工具简介 / 安装命令 / 核心命令 / 触发方式 / 截图目录 |
| 基本信息 | 工具组合 / 耗时 / context / 子代理使用 / 待办完成 |
| 技能使用 | 哪些 skill 被触发、触发方式、产出质量 |
| 业务规则覆盖 | 5 条规则逐条核对（已实现 / 遗漏 / 部分实现 / 跳过） |
| 遗漏与缺陷分析 | 重点检查"AI 自行降级"和"用户未参与"的决策 |
| 开发过程 | 工作流 / 人工干预 / 关键决策 |
| 产出质量 | 后端 / 前端 / 文档 |
| 上下文使用 | token 消耗 / 压缩 / 效率评估 |
| 工具体验 | 流畅度 / 自然度 / 安心感（1-5 分） |
| 用户评分 | N/10 + 加扣分理由 |
| 总结 | 一句话评价 + 数据速览 |

数据速览代码块格式（方便后期机读对比）：

```
耗时:       45m（wall time）
提交:       N 个
代码变更:   +XXX / -XX 行（N 文件）
测试:       N 个
文档:       N 行（N 份）
Context:    XXk / 200k（XX%）
人工干预:   N 次
业务规则:   N/5 完整，N 部分，N 跳过
```

## 其他类型的贡献也欢迎

- **修正笔记中的事实错误** —— 提 PR 说明哪里错了、依据是什么
- **补充新的对比维度** —— 如果你觉得"应该对比 X 但报告里没有"，欢迎单独提 PR 扩展 `comparison.md`
- **指出客观性偏差** —— 如果你发现报告中某个结论受了未声明的偏差影响，提 Issue 讨论
- **翻译** —— 当前笔记是中文，如果想做英文版可以提 PR 加 `notes/en/` 目录

---

## 不接受的贡献

- **不接受没跑过实验的"理论对比"** —— 这个仓库的核心价值是"亲手跑过的笔记"，没有实战的对比不在我们的目标范围
- **不接受"工具 A 比 B 好"这类没有数据支撑的结论** —— 必须基于具体数据点（耗时、token、代码 diff、规则覆盖）
- **不接受违反公平性原则的实验**：
  - 用了不同的需求（必须用 `requirement.md`）
  - 修改了 `base/`
  - 跨 session 共享上下文
  - 多次跑取最好结果（每个工具只能算一次，重复实验请另开轮次）

---

## 工作流约定

- 主分支：`main`
- 新一轮实验：建议在 fork 的分支上跑完再 PR，不要在本仓库直接 push
- Issue / PR 标题前缀建议：
  - `[round N]` 新一轮实验
  - `[notes]` 修笔记
  - `[comparison]` 改对比报告
  - `[meta]` 改 README / 贡献指南 / 仓库元信息
